# 🔐 Modern Login Page (UI/UX Study)

> Uma interface de autenticação elegante, desenvolvida com foco em Clean Design e usabilidade, utilizando HTML5 e CSS3.

## 📖 Sobre o Projeto

Este projeto faz parte do meu portfólio de estudos em **Programação Full-Stack**. Trata-se de uma página de login moderna que integra autenticação tradicional via formulário e opções de login social (Facebook e Twitter). 

O objetivo principal foi aplicar conceitos de design de interface (UI) que proporcionem uma experiência de usuário (UX) fluida, utilizando validações nativas do navegador e uma disposição de elementos intuitiva para garantir que o usuário finalize o acesso com o menor atrito possível.

## ✨ Destaques Técnicos

* **Validação Nativa:** Uso dos atributos `required` para garantir que os campos de e-mail/telefone e senha sejam preenchidos antes do envio.
* **Segurança Visual:** Implementação de campos do tipo `password` para mascarar a entrada de dados sensíveis.
* **Experiência de Usuário (UX):** Inclusão de funcionalidades essenciais como "Lembrar-me", recuperação de senha ("Esqueci a senha") e call-to-action para novos cadastros.
* **Iconografia Profissional:** Integração com a biblioteca Font Awesome (v5.15.2) para fornecer dicas visuais dentro dos campos de input (ícones de usuário e cadeado).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de formulários e containers.
* **CSS3:** Estilização avançada para layout centralizado e efeitos de hover.
* **Font Awesome:** Tipografia de ícones para auxílio visual na interface.
